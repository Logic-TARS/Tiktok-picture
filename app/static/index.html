新手说明

第一次使用按这个顺序走

  1. 确认你已经登录抖音创作者中心。
  2. 在“配置”里填 DeepSeek Key，模型默认可先用 deepseek-v4-flash。
  3. 在“素材”里选择文件夹或图片，系统会复制到本地工作目录。
  4. 确认“每组图片数”，需要改成 3 张或其他数量时，系统会自动按新规则重分组。
  5. 点击“创建任务”，系统会为每组图片生成标题、正文和话题。
  6. 点击任务里的“编辑”可以修改文案，页面会自动滚动到编辑区。
  7. 点击“半自动发布”后，系统会打开抖音创作者中心并尝试上传和填文案。
  8. 如果页面结构变化，先看下方运行日志和错误提示。
首次必看
  • DeepSeek Key 可以放在页面配置里，也可以放在项目根目录的 env.txt。
  • 如果页面提示 Playwright 或 FFmpeg 不可用，先用启动脚本重新启动。
  • 创建任务时，如果组数较多，文案生成会比 4 张一组更慢，这是正常的。
报错时怎么办
  • DeepSeek 未配置：检查页面中的 API Key 或 env.txt。
  • 创建任务很慢：通常是组数变多后在逐组生成文案，先看任务创建反馈卡片。
  • 编辑按钮没反应：现在会自动滚动到编辑区并高亮，如果没有跳转，刷新页面后重试。
  • 发布失败：看任务卡片中的原因说明和右侧运行日志。
