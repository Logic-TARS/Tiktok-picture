新手说明

第一次使用按这个顺序走

  1. 确认你已经登录抖音创作者中心。
  2. 在“配置”里填 DeepSeek Key，模型默认可先用 deepseek-v4-flash。
  3. 在“素材”里选择文件夹或图片 / 视频，系统会复制到本地工作目录。
  4. 在“发布类型”里选择图集或视频；视频支持直接发本地视频，也支持用当前图片自动混剪。
  5. 点击“创建任务”，系统会覆盖旧草稿任务并生成新的标题、正文和话题。
  6. 点击任务里的“编辑”可以直接在该任务下方修改文案。
  7. 点击“半自动发布”后，系统会按任务类型切到“发布图文”或“发布视频”。
首次必看
  • DeepSeek Key 可以放在页面配置里，也可以放在项目根目录的 env.txt。
  • 图片混剪视频依赖 FFmpeg，没有安装时无法创建混剪视频任务。
  • 再次点击“创建任务”会覆盖旧草稿任务，不会无限追加。
报错时怎么办
  • DeepSeek 未配置：检查页面中的 API Key 或 env.txt。
  • 混剪视频失败：通常是 FFmpeg 未安装或图片文件异常。
  • 视频发布失败：看任务卡片中的原因说明和右侧运行日志。
